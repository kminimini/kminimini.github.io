# 마크다운 문법 가이드

블로그 게시글 작성 시 사용할 수 있는 마크다운 문법을 소개합니다.

## 1. 제목
```
# 제목 1
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
###### 제목 6
```
# 제목 1
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
###### 제목 6

## 2. 강조
```
*이탤릭체*  
**볼드체**  
***이탤릭 볼드체***  
~~취소선~~  
<u>밑줄은 태그를 직접 사용.</u>
```
*이탤릭체*  
**볼드체**  
***이탤릭 볼드체***  
~~취소선~~  
<u>밑줄은 `<u></u>`태그를 직접 사용.</u>

## 9. 표
- 일반적인 표
  ```
  | 제목 1 | 제목 2 | 제목 3 |
  | ------ | --- | ---- |
  | 내용 1 | 내용 2 | 내용 3 |
  | 내용 1 | 내용 2 | 내용 3 |
  ```
  | 제목 1 | 제목 2 | 제목 3 |
  | ------ | --- | ---- |
  | 내용 1 | 내용 2 | 내용 3 |
  | 내용 1 | 내용 2 | 내용 3 |

- 정렬
  ```
  | 제목 1 | 제목 2 | 제목3 |
  | :--- | ---: | :---: |
  | 왼쪽정렬 | 오른쪽정렬 | 중앙정렬 |
  | 왼쪽정렬 | 오른쪽정렬 | 중앙정렬 |
  | 왼쪽정렬 | 오른쪽정렬 | 중앙정렬 |
  ```
  | 제목 1 | 제목 2 | 제목3 |
  | :--- | ---: | :---: |
  | 왼쪽정렬 | 오른쪽정렬 | 중앙정렬 |
  | 왼쪽정렬 | 오른쪽정렬 | 중앙정렬 |
  | 왼쪽정렬 | 오른쪽정렬 | 중앙정렬 |

- 셀 확장
  ```
  | 제목 1 | 제목 2 | 제목3 |
  | :--- | :---: | ---: |
  || 중앙에서 확장 ||
  ||| 오른쪽에서 확장 |
  | 왼쪽에서확장 ||
  ```
  | 제목 1 | 제목 2 | 제목3 |
  | :--- | :---: | ---: |
  || 중앙에서 확장 ||
  ||| 오른쪽에서 확장 |
  | 왼쪽에서확장 ||

- 셀 강조
  ```
  | 제목 1 | 제목 2 | 제목 3 |
  |------|---|----|
  | 내용 1 | 내용 2 | 내용 3 |
  | 내용 1 | **내용 2** | 내용 3 |
  | 내용 1 | <span style="color:red">내용 2</spen> | 내용 3 |
  ```
  | 제목 1 | 제목 2 | 제목 3 |
  |------|---|----|
  | 내용 1 | 내용 2 | 내용 3 |
  | 내용 1 | **내용 2** | 내용 3 |
  | 내용 1 | <span style="color:red">내용 2</spen> | 내용 3 |
